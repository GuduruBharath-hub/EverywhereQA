# everywhere-qa

The public CLI for [Everywhere QA](https://github.com/GuduruBharath-hub/EverywhereQA): deterministic accessibility, RTL, text-expansion, locale, and constrained-network audits with verified before/after Global Passports.

```sh
npx --yes everywhere-qa@0.2.0 setup
npx --yes everywhere-qa@0.2.0 audit --url https://example.com --out artifacts/everywhere-qa
```

See the repository README for configuration, Codex plugin installation, supported platforms, scoring, limitations, and the audit-fix-verify workflow.

MIT licensed. Automated evidence is not certification, legal advice, or a translation-quality assessment.
